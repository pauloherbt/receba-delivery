# MVP

+ Cadastrar Restaurante
+ Cadastrar Produtos
+ Cadastrar Categoria de produtos (facilitar o agrupamento)
+ Cadastrar cliente

## Entidades  

### Restaurante

+ Nome
+ Endereço
+ imagem
+ email_identificador
+ senha

### Produto

+ nome
+ preço
+ categoria
+ código interno
+ image_url
+ restaurante_id
+ category_id
+ createdAt
+ updatedAt

### Categoria

+ nome
+ restaurante_id
+ created_At
+ updated_At
